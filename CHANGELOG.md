- Initial Atomic Motion MVP.

- Parser and diagnostics.
- Dependency-free runtime scheduler.
- Initial fade/reveal style application.

- Runtime cleanup via `destroy()`.
- Motion validator CLI.
- Documentation, example page, and CI workflow.
